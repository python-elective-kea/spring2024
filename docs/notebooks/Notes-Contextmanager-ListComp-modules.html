Notes on Modules and files

Agenda of today:

  • how to read and write from/to files

    • text files

    • sqllite files

Files

Read from files

  1. The basic way

[3]:

f = open('testfiles/bohr.txt', 'r')
f.read()

[3]:

'An expert is a person who has made all the mistakes that can be made in a very narrow field.\nPrediction is very difficult, especially about the future.\nThose who are not shocked when they first come across quantum theory cannot possibly have understood it.\n'

  1. With the use of the contextmanager protocol

[10]:

with open('testfiles/bohr.txt', 'r') as f:
    print(f.read())
    # print(f.readline())
    # print(f.readlines())

An expert is a person who has made all the mistakes that can be made in a very narrow field.
Prediction is very difficult, especially about the future.
Those who are not shocked when they first come across quantum theory cannot possibly have understood it.

Write to files

  1. The basic way

[ ]:

f = open('testfiles/test.txt', 'w')
f.write('Hello world!')

  1. With the use of the contextmanager protocol

[11]:

with open('testfiles/test.txt', 'w') as f:
    f.write('Hello world')

2. 3rd party modules

2 step process
* install the module * import the module

Installation

Installation of a module is done by in the terminal typing:

pip install <modulename>

You can see all modules you have installed by using this command:

pip list

The Requests module

Have to be installed by:

Using a database (sqllite)

[ ]:

from sqlite3 import connect

conn = connect('testfiles/school.db')
cur = conn.cursor()
cur.execute('CREATE TABLE students(id int PRIMARY KEY, name text, cpr text)')
cur.execute('INSERT INTO students(id, name, cpr) VALUES (1, "Claus", "223344-5566")')
cur.execute('INSERT INTO students(id, name, cpr) VALUES (2, "Julie", "111111-1111")')
cur.execute('INSERT INTO students(id, name, cpr) VALUES (3, "Hannah", "222222-2222")')

for i in cur.execute('SELECT * FROM students'):
    print(i)

cur.execute('DROP TABLE students')
