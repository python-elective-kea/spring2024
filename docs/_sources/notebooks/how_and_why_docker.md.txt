# How and why we use docker


## The way we will use docker
I this elective we will use docker to execute your python scripts and your local machine (the host) for writing and editing code. 
This means that when it comes to writing code you can make use of your own operating system and use what ever editor you are used to or that you like to write your python scripts. 
When you want to execute the python script you just created you will sort of move it to another computer (the docker container) and execute it there. 

The reason for this procedure is that you are garanteed the same execution environment like everybody else that are in your group/class/team or even world.
