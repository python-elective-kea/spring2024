 6"""Words exercise
 7The main() below is already defined and complete. It calls print_words()
 8and print_top() functions which you write.
 91. For the --count flag, implement a print_words(filename) function that counts
10how often each word appears in the text and prints:
11word1 count1
12word2 count2
13...
14Print the above list in order sorted by word (python will sort punctuation to
15come before letters -- that's fine). Store all the words as lowercase,
16so 'The' and 'the' count as the same word.
172. For the --topcount flag, implement a print_top(filename) which is similar
18to print_words() but which prints just the top 20 most common words sorted
19so the most common word is first, then the next most common, and so on.
20Use str.split() (no arguments) to split on all whitespace.
21Workflow: don't build the whole program at once. Get it to an intermediate
22milestone and print your data structure and sys.exit(0).
23When that's working, try for the next milestone.
24Optional: define a helper function to avoid code duplication inside
25print_words() and print_top().
26"""

28import sys

30# +++your code here+++
31# Define print_words(filename) and print_top(filename) functions.
32# You could write a helper utility function that reads a file
33# and builds and returns a word/count dict for it.
34# Then print_words() and print_top() can just call the utility function.
35
36
37def util(filename):
38    d = {}
39    f = open(filename)
40    words = f.read().split()
41    
42    for i in words:
43        i = i.lower()
44        if i in d.keys(): 
45            d[i] += 1
46        else:
47            d[i] = 1
48    return d
49
50
51def print_words(filename):
52
53    x = dict(sorted(util(filename).items()))
54    for i, j in x.items():
55        print(f'{i}\t\t{j}')
56
57
58def print_top(filename):
59    
60    x =  list(util(filename).items())
61    x = sorted(x, reverse=True, key=lambda x: x[-1])
62    x = dict(x[:20])
63    
64    for i, j in x.items():
65        print(f'{i}\t\t{j}')
66

68# This basic command line argument parsing code is provided and
69# calls the print_words() and print_top() functions which you must define.
70def main():
71  if len(sys.argv) != 3:
72    print ('usage: python words.py {--count | --topcount} file')
73    sys.exit(1)
74
75  option = sys.argv[1]      # -> --count or --topcount
76  filename = sys.argv[2]    # -> the name of the file to read from
77
78  # call the function based on which option is specified
79  if option == '--count':
80    print_words(filename)
81  elif option == '--topcount':
82    print_top(filename)
83  else:
84    print ('unknown option: ' + option)
85    sys.exit(1)
