Each card has a letter on one face and a number on the other face.

Claim: If a card shows a vowel on one face, then its opposite face is an even number

Which card or cards must be turned over to test if this claim is true or false?

Your job is

  1. Make sure you understood the task

  2. Solve the challange

  3. Create an algorithm that for any 4 cards given can check if the cards can pass the test or not.

Then you should implement the algoritm and code the solution in this Jupyter Notebook. You should focus on using a pythonic coding style, and you should focus on making your application as fast as possible.
* first create a function that can generate 4 cards with an uppercase letter on one side and a single digit number on the orher side. * Then populate a list with a given number of card sets. * run each card set through the algorithm * get the result of how many card sets passed the test and how many did not, and specify which set violated the rule.

Generate a card set

[2]:

import random
import string
def generate():
    cards = []

    for i in range(4):
        s = random.choice(string.ascii_uppercase)
        d = random.randint(1, 9)
        r = random.randint(0, 1)

        if r == 0:
            cards.append((s, d))
        else:
            cards.append((d, s))

    return  tuple(cards)

Populate a list with card sets

[3]:

def populate(func, n):
    l = []
    for _ in range(n):
        l.append(func())
    return l

x = populate(generate, 10)

The Algorithm

[4]:

# if a card has a Vowel then we have to turn it
# if a card has a odd number we have to turn it

def check(l):
    vowels = ['A', 'E', 'I', 'O', 'U', 'Y']
    check = []

    for i in l:
        flag = False
        for j in i:
            if j[0] in vowels and j[1] % 2 == 1:       # a vowel and an odd number
                check.append((i, False, j))
                flag = True
                break
            elif type(j[0]) == int and j[0]%2 == 1 and j[1] in vowels:  # an odd number and a vowel
                check.append((i, False, j))
                flag = True
                break
        if not flag:
            check.append((i, True))

    return check

check(x)
