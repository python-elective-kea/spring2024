Example:

So when you use the len() build in function, you are calling the __len__() method of the object.

 1class Person:
 2  def __init__(self, name, height):
 3     self.name = name
 4     self.height = height
 5
 6  def __len__(self):
 7     return self.height
 8
 9x = Person('Claus', 180)
10len(x)    # this is calling/executing the __len__() method.

So when you use the + sign, you are calling the __add__() method of the object.

1class Person:
2     def __init__(self, height):
3       self.height = height
4     def __add__(self, other):
5       return Person(self.height/2 + other.height/2)
6
7x = Person(180)
8y = Person(162)
9x + y    # the + sign is calling/executing the __add__() method.
